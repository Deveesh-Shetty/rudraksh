# Technologies Behind Rudraksh

  ##  *Rudraksh leverages a robust stack of modern technologies to deliver a seamless experience for its users and maintain high accuracy in disease prediction.*

## Disease Prediction & Model Training:

- **[Python]**: The core language used for disease prediction, offering flexibility and efficiency.
- **[Google Colab]**: A powerful tool used for training our machine learning models in the cloud.
- **[TensorFlow]**: An open-source machine learning framework that aids in building and training the prediction models.
- **[Numpy]**: Essential for numerical computations in our prediction algorithms.
- **[Flask]**: A lightweight web framework that integrates our Python-based algorithms with our web frontend.
- **[PyCharm IDE]**: Our preferred integrated development environment for Python development, offering robust tools and features.

### Web Development & Deployment:

- **[Next.js]**: A React framework that enables server-rendered React applications, enhancing performance and user experience.
- **[TailwindCSS]**: A utility-first CSS framework used for crafting our web interface.
- **[Typescript]**: A typed superset of JavaScript ensuring robustness and better code quality.
